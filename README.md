# homebrew-tap

Custom Homebrew tap for formulas not available in the official Homebrew repositories.

## Installation

```sh
brew tap lmnotran/tap
```

## Formulas

| Formula | Description |
|---------|-------------|
| `bws` | [Bitwarden Secrets Manager CLI](https://bitwarden.com/help/secrets-manager-cli/) |

### Installing a formula

```sh
brew install lmnotran/tap/<formula>
```

For example:

```sh
brew install lmnotran/tap/bws
```
